HTML:

1: <div> → A container tag that groups element together and used here to keep the heading, textarea
and counter in one block.

2: <h2>Live Character Counter</h2> → A heading tag that displays the title.h2 is smaller than 
h1 used for subheadings.

3: <textarea id="textInput" placeholder="Start typing..." cols="60" rows="5"></textarea> →Creates a 
multi-line text box where the user can type. id="textInput" → gives it a unique name so JavaScript can 
target it.placeholder="Start typing..." → shows a hint until the user types.cols="60" → sets the width 
of the box in number of characters.rows="5" → sets the height of the box in number of lines.

4: <p>Characters: <span id="charCount">0</span></p> →Displays text “Characters:” followed by a counter.
<span> → inline element inside paragraph.id="charCount" → allows JavaScript to update this value.Initially
shows 0, meaning no characters typed.

5: </div> → Closes the container.


JavaScript

1: const textarea = document.getElementById("textInput"); →Finds the <textarea> using its id.Stores it in
the variable textarea for use in code.

2: const counter = document.getElementById("charCount"); →Finds the <span> using its id.Stores it in the
variable counter to update the number later.

3: textarea.addEventListener("input", () => { ... }); →Adds an event listener on the textarea."input" event
triggers whenever the user types, deletes, or pastes text.Runs the function inside { ... } every time input 
changes.

4: counter.textContent = textarea.value.length; → textarea.value → takes the text entered by the user.
length → calculates how many characters are typed.counter.textContent → updates the <span> to display 
the new count.
